### 6.2 ★ 必須要素 8 自動推定ロジック

#### 6.2.1 入力ソース

- `mb.description`(コンセプト記述)
- 全 `mb.items[].caption`(各画像のメモ)
- 連結 → 小文字化 → キーワードマッチング

#### 6.2.2 キーワード辞書(MVP・ヒューリスティック)

```typescript
const ESSENTIAL_KEYWORDS: Record<string, string[]> = {
  model:      ["[model]", "歳", "アンドロジナス", "細身", "印象", "雰囲気"],
  lifestyle:  ["[lifestyle]", "旅行", "都会", "孤独", "富裕", "芸術家", "ライフスタイル"],
  hair:       ["[hair]", "髪", "ヘア", "オールバック", "濡れ髪", "束ね"],
  makeup:     ["[makeup]", "メイク", "リップ", "アイライン", "ノーメイク"],
  clothes:    ["[clothes]", "服", "テーラード", "レザー", "ドレス", "コート", "シャツ"],
  light:      ["[light]", "光", "自然光", "逆光", "夕方", "朝日", "陰影"],
  location:   ["[location]", "海", "都市", "森", "廃墟", "室内", "海岸", "街"],
  color:      ["[color]", "黒", "白", "赤", "色", "モノクロ", "濃紺", "砂色"],
};

function detectEssentials(description: string, items: { caption: string }[]): Set<string> {
  const text = [description, ...items.map(i => i.caption)].join(" ").toLowerCase();
  const covered = new Set<string>();
  for (const [key, kws] of Object.entries(ESSENTIAL_KEYWORDS)) {
    if (kws.some(kw => text.includes(kw.toLowerCase()))) covered.add(key);
  }
  return covered;
}
```

- **プレフィックス `[hair]` 等を優先マッチ**(案 A2 のカテゴリ select で自動付与した場合は確実にマッチ)
